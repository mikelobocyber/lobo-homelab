# 🐺 Lobo Cybersecurity Homelab

## Overview

This repository documents my personal cybersecurity homelab built on Proxmox VE. The goal is to develop hands-on skills in virtualization, networking, penetration testing, and system administration through real-world lab environments.

## Objectives

* Build and manage a virtualized lab environment
* Develop practical cybersecurity skills
* Learn Linux system administration
* Practice penetration testing in a controlled environment
* Document technical work professionally

## Hardware

* GMKtec G3S Mini PC
* Intel N95 CPU
* 16GB RAM
* 512GB SSD

## Core Technologies

* Proxmox VE (bare-metal hypervisor)
* Kali Linux (attacker machine)
* Metasploitable (vulnerable target)
* Ubuntu Server (future services)
* Windows (planned for Active Directory lab)

## Current Lab Architecture

* Proxmox host system
* Kali Linux VM
* Metasploitable VM
* Internal isolated virtual network

## Skills Demonstrated

* Hypervisor installation and configuration
* Virtual machine provisioning
* Network isolation and lab design
* Linux system usage and management
* Troubleshooting and debugging
* Technical documentation
